## Што е ова?

Ова е систем за евиденција на присуство кој работи така што:

1. ✓ Професорот прикажува QR код на проектор
2. ✓ Вие го скенирате кодот со вашиот телефон/таблет
3. ✓ Системот автоматски ја проверува вашата локација
4. ✓ Доколку сте во училница, присуството се евидентира

### Зошто GPS?

GPS локацијата се користи за да се потврди дека **навистина сте присутни** во училницата. Не можете да скенирате од дома или од друго место.

## Што ви Треба

### Задолжително

- ✓ Паметен телефон или таблет
- ✓ Камера (за скенирање на QR код)
- ✓ GPS/Локација овозможена
- ✓ Интернет конекција (WiFi или мобилен интернет)
- ✓ Modern browser (Chrome, Safari, Firefox)

### Препорачано

- Полна батерија на телефонот
- Стабилна интернет конекција
- Ажуриран browser

## Прв Пат Користење

### 1. Дозволи во Browser Settings

Пред првото користење, треба да дозволите пристап до:

#### Камера

```
Chrome:   Settings > Privacy > Camera > Allow
Safari:   Settings > Camera > Allow
Firefox:  Permissions > Camera > Allow
```

#### Локација (GPS)

```
Chrome:   Settings > Privacy > Location > Allow
Safari:   Settings > Location Services > Safari > While Using
Firefox:  Permissions > Location > Allow
```

### 2. Тестирање

Пред час, тестирајте дали функционира:

1. Отворете го системот
2. Проверете дали browser бара дозволи
3. Кликнете "Allow" на секоја дозвола

## Скенирање на QR Код

### Правилно Држење

```
         📱
        / \
       /   \
      /     \
     /       \
    ─────────────
    Проектор
```

- **Растојание:** 2-5 метри од проектор
- **Агол:** Директно кон QR кодот
- **Стабилност:** Држете стабилно

### Совети за Подобро Скенирање

#### ✓ ДОБРО

- Стабилно држење на телефонот
- Директен поглед кон QR код
- Добро осветлување
- QR кодот целосно во рамка
- Користење на zoom доколку сте подалеку

#### ✗ ЛОШО

- Треперење на телефонот
- Премногу близу или далеку
- Рефлексии на проекторот
- Дел од QR кодот надвор од рамка
- Погрешен QR код

## Користење на Zoom Функцијата

### Кога да Користите Zoom

- Седите во задните редови
- QR кодот е премал
- Не можете да се приближите кон проектор
- Камерата не го фокусира добро кодот

### Како да Користите Zoom

```
┌─────────────────────┐
│                     │  [+] ← Zoom In
│   📷 Камера         │  2.0x
│      Преглед        │  [-] ← Zoom Out
│                     │
└─────────────────────┘
```

1. **Кликнете на [+] копчето** за да зумирате

   - Секое кликање го зголемува зумот за 0.5x
   - Може до максимален zoom (зависи од уред)

2. **Кликнете на [-] копчето** за да намалите

   - Секое кликање го намалува зумот за 0.5x
   - Минимум е 1.0x (нормален поглед)

3. **Тековниот zoom се прикажува** во средина
   - Пример: `1.0x`, `1.5x`, `2.0x`

### Совети за Zoom

- Почнете со 1.5x или 2.0x за задни редови
- Не преувеличувајте - многу zoom прави замаглување
- Држете стабилно кога користите zoom
- Доколку е замаглено, намалете малку

### ✗ Грешки при Скенирање

#### 1. TOO_FAR (Премногу Далеку)

```
┌─────────────────────────────────┐
│         ✗ Scan Failed           │
│                                 │
│ You are too far from the        │
│ classroom (> 50m)               │
│                                 │
│ Please move closer to the       │
│ professor's location            │
│                                 │
│ [Scan Again] [Change Info]      │
└─────────────────────────────────┘
```

**Што значи:** Вашата GPS локација е подалеку од 50 метри од професорот.

**Решение:**

- Приближете се кон училницата
- Проверете дали сте во вистинската просторија
- Доколку сте во училница, одете кон прозорец (подобар GPS)

---

#### 2. DUPLICATE_DEVICE (Дупликат Уред)

```
┌─────────────────────────────────┐
│         ✗ Scan Failed           │
│                                 │
│ This device has already been    │
│ used to scan for this session   │
│                                 │
│ Each device can only scan once  │
│                                 │
│ [Change Info]                   │
└─────────────────────────────────┘
```

**Што значи:** Со истиот уред веќе скенирале за оваа сесија.

**Решение:**

- Користете друг уред (телефон на другар)
- Обратете се до професорот за мануелно додавање
- Не пробувајте повеќе - нема да работи

---

#### 3. EXPIRED_TOKEN (Истечен Токен)

```
┌─────────────────────────────────┐
│         ✗ Scan Failed           │
│                                 │
│ The QR code has expired         │
│                                 │
│ Please scan the new QR code     │
│                                 │
│ [Scan Again]                    │
└─────────────────────────────────┘
```

**Што значи:** QR кодот е застарен (постар од 30 секунди).

**Решение:**

- Почекајте на новиот QR код на проекторот
- Скенирајте го новиот код
- Кодот се менува секои 30 секунди

---

#### 4. GPS_UNAVAILABLE (GPS Недостапен)

```
┌─────────────────────────────────┐
│         ✗ Scan Failed           │
│                                 │
│ Unable to get your GPS location │
│                                 │
│ Please enable GPS and try again │
│                                 │
│ [Scan Again]                    │
└─────────────────────────────────┘
```

**Што значи:** Системот не може да ја добие вашата GPS локација.

**Решение:**

1. Проверете дали GPS е овозможен на телефонот
2. Одете кон прозорец (подобар сигнал)
3. Излезете надвор на малку
4. Рестартирајте го браузерот
5. Обратете се до професорот

---

#### 5. INVALID_TOKEN (Невалиден Токен)

```
┌─────────────────────────────────┐
│         ✗ Scan Failed           │
│                                 │
│ Invalid or expired QR code      │
│                                 │
│ [Scan Again]                    │
└─────────────────────────────────┘
```

**Што значи:** QR кодот не е валиден за оваа сесија.

**Решение:**

- Проверете дали скенирате го правилниот QR код
- Проверете дали професорот има активна сесија
- Скенирајте го новиот код

## Често Поставувани Прашања

### Општи Прашања

**П: Дали мора да имам интернет?**
О: Да, интернет конекција е задолжителна. WiFi или мобилен интернет.

**П: Дали мора да имам GPS?**
О: Да, GPS е задолжителен за верификација на локација.

**П: Колку често можам да скенирам?**
О: Само еднаш по сесија со секој уред.

**П: Дали можам да скенирам за другар?**
О: НЕ. Секој студент мора сам да скенира со свој уред на своја локација.

**П: Дали податоците се зачувуваат?**
О: Да, вашите информации се зачувуваат локално за полесно пополнување следниот пат.

---

### Прашања за Скенирање

**П: Колку време имам да скенирам?**
О: Додека професорот не ја заврши сесијата. Обично во текот на часот.

**П: Што ако не успеам да скенирам?**
О: Обратете се до професорот. Тој може да ве додаде мануелно.

**П: QR кодот се менува - кој да скенирам?**
О: Скенирајте го тековниот код на екранот. Тој е валиден 30 секунди.

**П: Дали можам да фотографирам QR код?**
О: НЕ. Кодот е валиден само 30 секунди и не работи од фотографија.

**П: Зошто zoom не работи?**
О: Zoom работи само на уреди кои поддржуваат camera zoom. Доколку не го гледате, вашата камера не го поддржува.

---

### Прашања за GPS

**П: Зошто ми треба GPS?**
О: За да се провери дека навистина сте присутни во училницата.

**П: Дали професорот ја гледа мојата точна локација?**
О: Не, се гледа само растојанието до професорот, не точната локација.

**П: Колку точен е GPS?**
О: GPS има точност од 5-20 метри. Може да варира во зависност од уред и локација.

**П: GPS не работи во зграда**
О: Одете кон прозорец за подобар сигнал. Доколку не работи, обратете се до професорот.

---

### Проблеми и Грешки

**П: Грешка "TOO_FAR" но сум во училница**
О: GPS понекогаш не е точен. Одете кон прозорец, почекајте 30 секунди за подобар сигнал.

**П: Грешка "DUPLICATE_DEVICE"**
О: Веќе сте скенирале со овој уред. Користете друг уред или обратете се до професорот.

**П: Камерата не работи**
О: Проверете browser permissions. Рестартирајте го браузерот. Користете Chrome.

**П: Системот е бавен**
О: Проверете интернет конекција. Затворете непотребни апликации. Рестартирајте телефон.

## Проблеми и Решенија

### Камера не Работи

**Симптоми:**

- Црн екран наместо камера
- Порака "Camera access denied"
- Камерата не стартува

**Решенија:**

1. **Провери Permissions**

   ```
   Chrome:
   Settings > Site Settings > Camera > Allow

   Safari:
   Settings > Safari > Camera > Allow

   Firefox:
   Permissions > Camera > Allow
   ```

2. **Рестартирај Browser**

   - Затвори го комплетно браузерот
   - Отвори го повторно
   - Обиди се повторно

3. **Пробај Друг Browser**

   - Chrome (препорачано)
   - Safari (iPhone)
   - Firefox

4. **Рестартирај Телефон**
   - Исклучи го телефонот
   - Почекај 10 секунди
   - Вклучи го повторно

---

### GPS не Работи

**Симптоми:**

- Порака "Getting location..." задолго
- Порака "GPS unavailable"
- Грешка "TOO_FAR" во училница

**Решенија:**

1. **Овозможи GPS**

   ```
   Android:
   Settings > Location > On

   iPhone:
   Settings > Privacy > Location Services > On
   ```

2. **Дозволи во Browser**

   ```
   Провери дали браузерот има дозвола за локација
   ```

3. **Оди кон Прозорец**

   - GPS подобро работи кај прозорец
   - Избегнувај подземја или центар на зграда

4. **Почекај Подолго**

   - Прв пат може да потрае 30-60 секунди
   - Држи го телефонот стабилно

5. **Излези Надвор**
   - Доколку е можно, излези надвор на кратко
   - GPS ќе се фиксира побрзо

---

### QR Код не се Скенира

**Симптоми:**

- Камерата работи но не детектира QR
- Ништо не се случува при скенирање
- Грешка "Invalid QR code"

**Решенија:**

1. **Провери Фокус**

   - Камерата треба да го фокусира кодот
   - Додај или намали растојание

2. **Користи Zoom**

   - Кликни [+] за да зумираш
   - QR кодот треба да биде јасен

3. **Провери Осветлување**

   - Избегнувај рефлексии на проектор
   - Промени агол доколку има рефлексија

4. **Провери Стабилност**

   - Држи стабилно
   - Не треперај

5. **Скенирај Нов Код**
   - Почекај на новиот QR (30 сек)
   - Пробај повторно

---

### Други Проблеми

#### Бавен Систем

- Затвори непотребни апликации
- Провери интернет брзина
- Рестартирај телефон

#### Интернет Конекција

- Провери дали WiFi работи
- Користи мобилен интернет

#### Батерија

**Решенија:**
